Edubot

Edubot is an Educational Robot, which is operated by Bangla & English scratch programming. The main purpose of this robot is mainly to teach kids programming logic & basic robotics in a gamified way. It was a Bangladesh govt & international Investor funded project.
This whole robot's software & firmware architecture was developed by me. In v1.0.0 to v2.0.0 processor was atmega328p & v3.0.0 to v4.2.0 processor was esp32. The communication system was Bluetooth & WiFi. Bluetooth was used for uploading user code(scratch programming) & WiFi was used for OTA(firmware updating). The firmware is written for dynamic scratch programming so that users can write any kind of scratch programming as they want to do (like basic Science projects, line follower robot, obstacle avoider robot, maze solver, music making, and so many robotic projects based on various types of sensors on it). It has sensors reading section(IR sensors, Ultrasonic sensor, Temperature sensor, LDR sensor, Gas sensor, and USB plugin-play type sensor), actuators writing section(Motor, RGB Led, Buzzer), making music, variable creation, if else, while loop, for loop, etc.
